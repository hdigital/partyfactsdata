# Development environment

## Codespaces

The project can use [GitHub Codespaces](https://github.com/features/codespaces) as a development environment.

Dependencies (e.g., R packages, Python tools) are installed on build.

### Usage

RStudio Server starts automatically and is accessible on port 8787 via the Ports tab.

Login credentials:

- Username: `rstudio`
- Password: `rstudio`

To manually control RStudio:

```bash
sudo rstudio-server start
sudo rstudio-server stop
```
